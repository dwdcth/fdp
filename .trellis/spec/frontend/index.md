# Frontend Development Guidelines

> Best practices for frontend development in this project.

---

## Overview

This repository currently has no frontend implementation. These files exist to document that reality and to prevent AI assistants from inventing browser-side architecture.

## Pre-Development Checklist

Before adding any frontend code:

1. Read [Directory Structure](./directory-structure.md)
2. Read [Component Guidelines](./component-guidelines.md)
3. Read [State Management](./state-management.md)
4. Read [Type Safety](./type-safety.md)
5. Update these guides immediately after choosing a real frontend stack

---

## Working Rule

Do not infer React/TypeScript conventions from the existence of these template files. Until a frontend exists, repository reality is the standard.
